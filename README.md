# Medir Conexão
App de testes de internet construído para testar conhecimentos de Tailwind, o layout e simples utilizando o OpenSpeedTest como medidor

## Funções
 - Testar internet
 - Mostrar resultados
 - Layout Light/Dark
 - Foco em usabilidade e desempenho
